Manifestation Prompt Oracle — How to Use

1) Download and unzip the folder.
2) Open index.html on your phone or computer.
   - iPhone: Files → tap index.html → opens Safari
   - Android: Files → tap index.html → opens Chrome
3) Tap “Draw Prompt” to receive a daily manifestation prompt.
4) Choose a category: All / Money / Love / Self / Healing.
5) Optional (App-like use):
   - iPhone Safari → Share → Add to Home Screen
   - Android Chrome → Menu → Add to Home Screen

Works offline. No app needed.
